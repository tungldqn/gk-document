# GK Document

Kho tài liệu mô tả nghiệp vụ và yêu cầu tính năng cho dự án **GK** (Quản lý bài kiểm tra).

Mỗi tài liệu phân tích một tính năng cụ thể, bao gồm: mô tả tổng quan, luồng nghiệp vụ, quy tắc xử lý, trạng thái giao diện và các trường hợp đặc biệt.
